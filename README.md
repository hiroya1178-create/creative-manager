# Creative Manager Public App v2.3

GitHub Pages にそのままアップロードできる公開用4ファイル版です。

## 今回までに入っているもの
- ダッシュボード
- 受注管理
- スタッフ管理（追加 / 編集 / 削除）
- テンプレート管理（追加 / 編集 / 削除）
- テンプレートから案件作成
- 新規案件追加
- PDF発注書読み込み候補入力
- 日程カレンダーから案件編集
- 外注管理の実務向け編集

## 使い方
`index.html` / `style.css` / `app.js` / `README.md` の4ファイルを
GitHub リポジトリのルートにアップロードしてください。
