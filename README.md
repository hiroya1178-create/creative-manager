# Creative Manager Public App v2.2

GitHub Pages にそのままアップロードできる公開用4ファイル版です。

## 今回の追加
- 新規案件追加で PDF 発注書の添付
- PDF 内の文字を読み取り
- 顧客名 / 案件名 / 金額 / 完了予定日の候補を自動入力

## 入っている画面
- ダッシュボード
- 受注管理
- スタッフ管理
- テンプレート管理
- 日程カレンダー
- 外注管理

## 使い方
`index.html` / `style.css` / `app.js` / `README.md` の4ファイルを
GitHub リポジトリのルートにアップロードしてください。
